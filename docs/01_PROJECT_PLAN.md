# Bible eBook Project Plan
 
## Goal
Create a premium Bible handwriting eBook archive for church members.
 
## Requirements
- Web + Mobile responsive
- HTML static architecture
- 66 Bible books with individual covers
- Flipbook experience (book-like page turning)
- Writer search (index by person)
- Full table of contents
- QR-first distribution
- FlipHTML5-inspired premium UX
 
## 3 Main Views
1. Landing (intro + full Bible covers)
2. Reader (flipbook)
3. Gallery (making-of archive)
 
## Pipeline
PDF scan -> OCR/index -> WEBP conversion -> Static generation -> Deploy
